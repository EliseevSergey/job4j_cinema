**job4j_cinema**

Pet project on job4j course.

Cinema site with pages for:<br>
-Registration/Login<br>
-List of movies and sessions<br>
-Ticket purchasing


**Stack:**
Java 18, SpringBoot 2.7.6, Thymeleaf, Bootstrap,
Maven 4.0.0, Liquibase 4.15.0, Sql2o 1.6.0, h2 2.1.214
PostgreSQL 42.5.1

**Requirements**<br>
Java 18, SpringBoot 2.7.6, Thymeleaf, Bootstrap,
Maven 4.0.0, Liquibase 4.15.0, Sql2o 1.6.0, h2 2.1.214
PostgreSQL 42.5.1

**Installation**<br>
Run Main class
